# Future External Clients

Future clients may include CLI users, TUI users, MCP clients, editor panels, service agents, and optional runtime programs.

From a product point of view, these clients should get the same durable semantics as Pi users:

- read graph-backed status,
- inspect current knowledge and roadmap state,
- request compiler workflows,
- record evidence and validation outcomes through typed capabilities,
- avoid editing generated files directly.

Technical distribution details belong under system clients.
